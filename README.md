# RayFlag - Xray Staffer Flag
## Commands
- /raycheck [nickname player ] - Controlla tutte le flag history giornaliere di tutti i giocatori che hanno mandato delle flag

## Permissions
- rayflag.flag - Permesso per poter visionare le flag in "diretta" quando un giocatore rompe un minerale
- rayflag.checkflag - Permesso per poter eseguire il comando della history e visionare la history delle flag di un preciso giocatore

*Materiali considerati **Materiali xray**:*
- Diamante (diamond_ore)
- Diamante Profondo (deepslate_diamond_ore)
- Oro (gold_ore)
- Oro Profondo (deepslate_gold_ore)
- Smeraldo (emerald)
- Smeraldo Profondo (deepslate_emerald_ore)

  ### Ulteriori materiali arriveranno con nuovi snapshoot o aggiornamenti!
